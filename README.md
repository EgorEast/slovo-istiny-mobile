# Mobile ap for church on React Native based on Feature Sliced Design

🍰 [Architecture design guidelines](https://feature-sliced.design/)

## Technology stack

- **UI**: `react`, `react-native`, `expo`,
- **Data model**: `zustand`,
- **Requests**: `ky`,
- **Lang**: `typescript`
- **Lint**: `prettier`, `eslint`
- **Architecture**: `feature-sliced`
